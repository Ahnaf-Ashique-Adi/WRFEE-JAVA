# Retro Runner

A simple retro-style infinite runner game built in Java.

## Prerequisites

- Java Development Kit (JDK) 8 or higher.

## Setup and Compilation

1.  Open a terminal or command prompt.
2.  Navigate to the project root directory.
3.  Compile the source code using the following command:

    ```bash
    javac -d bin src/*.java
    ```

    This will compile all Java files in the `src` directory and place the class files in the `bin` directory.

## Running the Game

To run the game, execute the following command from the project root directory:

```bash
java -cp bin game
```

## Controls

-   **W / Up Arrow**: Move Up
-   **S / Down Arrow**: Move Down
-   **A / Left Arrow**: Move Left
-   **D / Right Arrow**: Move Right
-   **Space**: Start Game / Restart Game
